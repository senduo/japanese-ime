How to generate bigram source files:

1. Download Japan wiki dump files of choise from:
http://dumps.wikimedia.org/jawiki/*/*pages-articles*.xml.bz2

2. Install:
wp2txt: https://github.com/yohasebe/wp2txt
kytea: http://www.phontron.com/kytea/

3. Create a temp dir

4. Create src in the temp dir and copy the files in the wiki bz2 archive to it

5. Execute conv.sh in the temp dir to generate files in [temp dir]/dest

6. Modify BigramSrcGen to load the files from the dest dir and run it

7. unigram-freq.txt and bigram-freq.txt will be generated in target dir
